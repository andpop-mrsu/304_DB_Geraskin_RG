Лабораторная работа 8. Сценарии PHP. Разработка CRUD-приложения для работы с БД



Описание проекта
Веб-приложение предназначено для ведения базы данных студентов и учета результатов их экзаменационных сессий. Программа реализует полный цикл CRUD (создание, чтение, обновление, удаление) для управления персональными данными учащихся и их успеваемостью.



Запуск приложения

Запустите локальный веб-сервер из каталога Task08 командой:
php -S localhost:3000 -t public

Откройте приложение в браузере по адресу:
http://localhost:3000/
